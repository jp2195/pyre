# View Reference

One page per view. Use [keybindings.md](../keybindings.md) for the
canonical key reference; the per-view pages here cover what each view
shows, what its filter / sort fields are, and what the detail panel
reveals.

## Monitor (`1`)

- [Dashboard](dashboard.md) — system, resources, HA, network,
  security, VPN at-a-glance

## Analyze (`2`)

- [Policies](policies.md) — security rules with hit counts
- [NAT](nat.md) — NAT rules with hit counts
- [Objects](objects.md) — address & service objects
- [Sessions](sessions.md) — active sessions
- [Interfaces](interfaces.md) — interface status, counters, errors
- [Routes](routes.md) — routing table + BGP / OSPF neighbors
- [IPSec](ipsec.md) — site-to-site IPSec tunnel status
- [GP Users](gpusers.md) — GlobalProtect connected users
- [Logs](logs.md) — system / traffic / threat logs

## Tools (`3`)

- Config dashboard — rule summary + pending changes. See
  [Dashboard](dashboard.md).
